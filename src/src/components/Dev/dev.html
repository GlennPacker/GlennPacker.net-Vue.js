Install and Quick Start

Download Vue Javascript from
https://unpkg.com/vue

As a script tag
<script type="text/javascript" src="https://unpkg.com/vue"></script>

A great quick start project with lots of extras is found on git hub . This project comes with:

  • Vue.js 2
  • Vue Router 2
  • Axios
  • Animate.css
  • Babel
  • Bootstrap 4
  • BrowserSync
  • ESLint
  • Font Awesome
  • JSONPlaceholder
  • SASS
  • Webpack 2
  • Yarn
