Components

The basic component

import Vue from 'vue';
import template from './template.html';

export default Vue.extend({
  template,
});

The template should be the html for the view/partial with a single root html element often a div.

Component With data and methods

import Vue from 'vue';
import template from './template.html';

export default Vue.extend({
  template,

  data(){
    return {
      message: 'hi'
    };
  },

  methods:{
    talk(){
      console.write(this.message);
    }
  }
});

There are many built in methods/funcitons built in (including computed, lifecycle handling), full docuemntation is found on the Vue JS website.
